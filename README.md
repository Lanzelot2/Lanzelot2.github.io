# Modrinth: Modpack installieren (EbeluCraft)

Auf dem EbeluCraft-Server kann man zwar komplett **ohne** Mods spielen, aber aus Performance- und Gameplay-Gründen (z. B. Ingame-Voice-Chat) ist es klar empfohlen, mit dem Modpack zu spielen.

> **Kurz gesagt:** Modrinth App installieren → einloggen → RAM einstellen → Modpack importieren → Minecraft starten → Server beitreten.

---

## Voraussetzungen

- Ein normal installiertes Minecraft (Java Edition)
- Windows / macOS / Linux
- **Mac-User:** Achtet später darauf, dass der Launcher Zugriff aufs Mikrofon bekommen kann (Voice-Chat).

---

## Schritt-für-Schritt

1. **Modrinth App installieren**  
   Geht auf die offizielle Seite und ladet euch die App runter: https://modrinth.com/app  
   Installiert sie wie jede andere App.

2. **In euren Minecraft-Account einloggen**  
   Öffnet Modrinth, klickt links auf **Account** und meldet euch an. Ihr werdet kurz in den Browser weitergeleitet und folgt den Schritten dort.

## Zusatz: Mods verwalten

Im Profil unter **Content** seht ihr alle installierten Mods. Dort könnt ihr:
- Mods updaten
- Mods deaktivieren (falls etwas stört)
- Optional deaktivierte Mods aktivieren, wenn ihr mehr Features wollt

<details>
<summary>Wenn Minecraft nicht startet / laggt</summary>

- Prüft zuerst die RAM-Einstellung (Schritt 4). Zu wenig oder viel zu viel kann Probleme machen.  
- Schließt andere RAM-fressende Programme (Browser-Tabs, Discord-Streams, etc.).  
- Falls ihr auf macOS seid: Mikrofon-Berechtigungen prüfen (für Voice-Chat).

</details>
