Auf dem EbeluCraft Server ist es zwar möglich komplett **ohne** Mods, jedoch ist es aus Performance und Gameplay technischen Gründen (Ingame Voice Chat) stärkstens empfohlen mit einigen Mods zu spielen

Um die Mods leichter verwalten und installieren zu können, empfehle ich zwar die Modrinth App.

Jedoch ist für technisch versierte auch jeder andere Launcher erlaubt, der Modded Minecraft unterstützt und, vor allem, für die, die auf einem Mac spielen, Zugriff auf das Mikrofon bekommen kann 

Alle die die Modrith App schon instaliert haben, können direkt zu Schritt 3 gehen.  

Als erstes installieren wir die [Modrinth App](https://modrinth.com/app). Ihr instaliert die App wie jede andere app auch.

Noch als zusatzt Info:

Innerhalb des Profils sieht man unter "Content" auch die verschiedenen Mods, die installiert sind. Von dort aus können auch neue installiert bzw. alte geupdatet oder auch aktive deaktiviert werden, falls sie stören sollten oder nciht gewollt sind. Außerdem befinden sich in dem Profil einige deaktivierte Mods, die man falls erwünscht, aktivieren kann, um weitere Features zu bekommen.
